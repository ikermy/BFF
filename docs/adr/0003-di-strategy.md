# ADR 0003: DI Strategy

- Статус: accepted
- Дата: 2026-03-17

## Решение

Используем ручную композицию зависимостей в `cmd/*` и `internal/app`.

## Причины

- Прозрачный граф зависимостей и простая отладка.
- Нет runtime-контейнера и скрытой магии.
- Удобно для текущего размера проекта.

## Пересмотр решения

Вернуться к вопросу при существенном росте числа модулей и провайдеров.
